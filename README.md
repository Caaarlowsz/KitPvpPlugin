# SGPvP Plugin

### Features
Soup PvP \
Custom messages \
Game events (grace period, feast, deathmatch) \
25+ kits \
Spectate on death \
Kit UI

### Game states
Game start \
Grace period \
Feast spawn \
Deathmatch \
Server close
